\section*{Abstract}
Brain activity unfolds over hierarchical timescales that reflect how brain regions integrate and process information, linking functional and structural organization. While timescale studies are prevalent, existing estimation methods rely on the restrictive assumption of exponentially decaying temporal autocorrelation and only provide point estimates without standard errors, limiting statistical inference. In this paper, we formalize and evaluate two methods for mapping timescales in resting-state fMRI: a time-domain fit of an autoregressive (AR1) model and an autocorrelation-domain fit of an exponential decay model. Rather than assuming exponential autocorrelation decay, we define timescales by projecting the fMRI time series onto these approximating models, requiring only stationarity and mixing conditions while incorporating robust standard errors to account for model misspecification. We introduce theoretical properties of timescale estimators and show parameter recovery in realistic simulations, as well as applications to fMRI from the Human Connectome Project. Comparatively, the time-domain method produces more accurate estimates under model misspecification, remains computationally efficient for high-dimensional fMRI data, and yields maps aligned with known functional brain organization. In this work, we show valid statistical inference on fMRI timescale maps, and provide Python implementations of all methods.

\section*{Appendix: Autocorrelation-Domain Score Process}

This appendix derives the score process $u_t$ from equation \eqref{eq:ad-qomega}. We omit scaling factors (e.g., $1/K$), as they cancel out in the sandwich variance ratio $\qad^{-1} \omegaad \qad^{-1}$.\\

\noindent The pseudo-true parameter $\phiad^*$ minimizes the population objective function $S_K(\phiad)$ \eqref{eq:nlm_loss}, satisfying the first-order condition:
\begin{align}\label{eq:a1}
    \sum_{k=1}^{K} \psi_k(\phiad^*) (\rho_k - \phiad^{*k}) = 0, \quad \text{where } \psi_k(\phiad^*) = \left. \frac{d}{d\phiad}(\phiad^k) \right|_{\phiad=\phiad^*} = k\phi_{\text{AD}}^{*k-1}.
\end{align}
Using $\rho_k = \mathbb{E}[X_t X_{t-k}]/\gamma_0$ \eqref{eq:acf}, we rewrite the residual term as an expectation:
\begin{align}\label{eq:a2}
    \rho_k - \phiad^{*k} = \mathbb{E}\left[ \frac{X_t X_{t-k} - \phiad^{*k} X_t^2}{\gamma_0} \right].
\end{align}
Substituting \eqref{eq:a2} into \eqref{eq:a1} defines the score process $u_t$:
\begin{equation}
    \mathbb{E}\left[ \frac{1}{\gamma_0} \sum_{k=1}^{K} \psi_k(\phiad^*) (X_t X_{t-k} - \phiad^{*k} X_t^2) \right] = \mathbb{E}[u_t] = 0.
\end{equation}
This matches the definition in equation \eqref{eq:ad-qomega}. The asymptotic variance is then determined by the long-run variance of this score, $\omegaad = \sum_{\ell=-\infty}^{\infty} \mathbb{E}[u_t u_{t-\ell}]$.\\

\noindent The sample score process $\hat{u}_t$ used for variance estimation is obtained by replacing population expectations with sample moments and the parameter $\phiad^*$ with the estimator $\hatphiad$:
\begin{equation}
    \hat{u}_t = \frac{1}{\hat{\gamma}_0} \sum_{k=1}^{K} k (\hatphiad)^{k-1} (x_t x_{t-k} - (\hatphiad)^k x_t^2).
\end{equation}
